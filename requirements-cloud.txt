#   - opencv-python-headless kept for image preprocessing in ocr_processor.py.
#
# OCR strategy on cloud:
#   1. Gemini Vision API (primary — needs GEMINI_API_KEY)  ← always used when key set
#   2. Local OCR skipped gracefully if package not installed
